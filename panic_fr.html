Comment ça marche?

Quels horaires sont affichés ?
Par défaut, l'application affiche les horaires dans les deux heures à venir. En mode temps réel, sont indiqués les horaires de l'heure à venir.
Voir les horaires précédents ou suivants
Il est possible de taper deux fois sur le côté gauche (ou droit) pour voir les horaires précédents (suivants)
Voir les horaires à une autre date
Il est possible de sélectionner une autre date via le bouton horloge
Que représente la flèche sur les favoris ?
la flèche indique la direction générale (Ouest, Est,...) du voyage dont l'heure de passage figure à droite.
Écart entre départ et arrivée
Dans certains cas, il peut y avoir un écart entre heure d'arrivée à un arrêt et heure de départ. Il est possible de choisir globalement l'affichage de l'un ou l'autre via les préférences.
Dans la grille principale, il est aussi possible de basculer via un appui long.
Voir son heure d'arrivée
Sur la grille, il est possible d'appuyer sur une heure de passage pour obtenir les horaires du reste du voyage
Horaires temps réel
Sur la grille, il est possible d'afficher les horaires temps réel via le bouton central. (nécessite un accès internet)
Ordre des arrêts
Les arrêts sont uniquement affichés dans l'ordre alphabétique, car faire autrement peut être un plus complexe qu'il n'y parait (par exemple, certaines lignes peuvent avoir des trajets plus ou moins différents pour une raison ou une autre)
